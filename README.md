# NoSQL-Social-Network-API

## Description

This is a NoSQL based API for a social network web application where users can share their thoughts, react to friendsâ€™ thoughts, and create a friend list. Express.js was used for routing while a MongoDB database and the Mongoose ODM are used to serve and store data.
